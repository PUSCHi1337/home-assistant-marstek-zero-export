# v0.1.0

Initial release of the Home Assistant Marstek Zero Export project.

## Included

- Home Assistant automation blueprint for Marstek Venus zero-export control.
- Optional helper package for the required input booleans and input numbers.
- Lovelace dashboard example.
- Setup, entity mapping, safety, and troubleshooting documentation.
- MIT license.

## Tested

- Blueprint import tested with Home Assistant `2026.5.4`.
- Disabled blueprint automation instantiation passed Home Assistant config check.
- Control strategy field-tested on one real Marstek Venus setup.

## Notes

This is an early `v0.1.x` release. Start with conservative power limits and observe the system during the first live test.

Dynamic tariff charging and grid charging are intentionally out of scope for this repository.
